# Cornerstone Christian Academy starter site

This is a lightweight GitHub Pages starter site for Cornerstone Christian Academy.

## Quick publish
1. Create a repo named `academy`
2. Upload all files in this folder to the repo root
3. In GitHub: Settings -> Pages -> Deploy from branch -> main -> /(root)
4. Your site will publish at `https://crstoneacademy.github.io/academy/`

## Replace placeholders
- `/assets/logo-placeholder.svg`
- `/assets/favicon.svg`
- contact email in `contact.html`
